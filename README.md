# ⭐ Why this repository?

Finding trustworthy Windows optimization software is difficult.

There are hundreds of scripts and "gaming optimizers" on the internet.

Some improve Windows.

Some are outdated.

Some are unsafe.

This repository only includes trusted projects that are widely used by the Windows community.
